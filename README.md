# ⚽ REAL FOOTBALL

Versão beta do meu jogo de futebol 2D para dois jogadores, desenvolvido puramente com **HTML5 Canvas** e **JavaScript**.

## 🎮 Como Jogar
Acesse o link do jogo aqui: https://arthurcruzz17.github.io/real_football/

### Controles:
- **Jogador 1 (W,A,D):** Movimentação e Pulo.
- **Jogador 2 (Setas):** Movimentação e Pulo.

## 🚀 Tecnologias Utilizadas
- **HTML5:** Estrutura e Canvas.
- **CSS3:** Estilização e interface.
- **JavaScript (ES6):** Motor de física, lógica de colisão e estados de jogo.

## ⚙️ Funcionalidades
- Sistema de colisão customizado.
- Placar dinâmico com identificação de cores.
- Cronômetro de partida com alerta visual de tempo esgotado.
- Condição de vitória por limite de gols (10) ou tempo.
- Traves com profundidade visual e física no travessão.
